**Лабораторная работа №8**

**Цель работы:**  
Приобретение практических навыков диагностики работы программного обеспечения.

**Задание:**  
В ходе выполнения лабораторных работ по курсу «Операционные системы» необходимо продемонстрировать ключевые системные вызовы, используемые в программах, и подтвердить, что их применение соответствует варианту лабораторной работы.

По итогам выполнения всех лабораторных работ отчёт по данной ЛР должен содержать краткую сводку по исследованию написанных программ.

**Средства диагностики:**

**Для ОС *nix (Linux/Unix):**
- **strace**
